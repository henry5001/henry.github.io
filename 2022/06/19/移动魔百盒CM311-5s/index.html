移动魔百盒CM311-5s安装第三方桌面

前言

移动魔百盒CM311-5s，支持wifi，蓝牙，蓝牙语音遥控，但是盒子生态过于封闭，目前还没有适配的刷机包，所以有了安装第三方桌面的想法。

商品名称：魔百和CM311-5s
商品编号：100035117726
商品毛重：200.00g
商品产地：中国大陆
功能：Wi-Fi，教育CPU
核心数：四核
网口规格：百兆网口
投屏协议：不支持
专属模式：其他
Wi-Fi连接：双频Wi-Fi 5
内容源：其他
输出分辨率：1080P
遥控器：红外遥控
恢复出厂设置的重置密码：10086

准备工作与思路

能开热点的windows、mac或linux系统，因为需要利用hosts文件的特性，将配置的域名解析到hosts指定的IP地址。

利用盒子播放某些视频时会请求固定的域名触发自动下载安装咪咕极速.apk的特性。

将gamedownload.taipan.bja.bcs.ottcn.com域名解析到新指定的IP地址175.24.235.166

这个新ip目录下含有楼主设置的一个当贝应用.apk文件

原本对gamedownload.taipan.bja.bcs.ottcn.com的解析将会由咪咕极速版.apk，转变为当贝应用.apk

开始工作与实践

（1）修改电脑hosts文件

需要管理员身份打开hosts文件，具体路径：C:\Windows\System32\drivers\etc

保存后记得在cmd使用命令刷新ipconfig /flushdns

（2）使用安卓手机连接测试

使用安卓手机连接修改hosts完毕后电脑开的热点并访问链接http://gamedownload.taipan.bja.bcs.ottcn.com/upgrade/nams/app/1269/1649925894696migujisu-beijing.apk

将会得到一个16M大小的apk文件，查看安装包图标或安装后确认是当贝应用.apk，说明前面的hosts修改已经生效。
（3）使用魔百盒复现手机连接的测试结果
首先，确保之前魔百盒内没有安装咪咕极速，可以使用遥控蓝牙语音口令卸载咪咕极速将咪咕极速卸载。
后面可能要看脸了，因为要访问一个能触发下载安装咪咕极速（那个域名）的视频，或者说你记得有哪些视频可以触发咪咕极速的下载，也可以直接搜索后访问，等安装咪咕极速的提示走完，顺利的话你的魔百盒内已将当贝应用安装进去。

（4）配置与使用

遥控蓝牙语音指令打开当贝桌面，即可进入当贝桌面。

遥控蓝牙语音指令打开桌面，也可以点击遥控的主页按钮，即可进入桌面选择，选择原装系统桌面和当贝桌面。

结语

感谢原楼主提供的当贝桌面资源ip服务，大家快去白嫖。^1

我们可以自己抓取电视盒子访问安装软件的请求地址，根据抓来的请求地址架设一套服务，修改hosts解析到架设的ip服务，再请求域名解析访问到修改hosts后架设的ip服务即可。

外接USB存储设备

FAT32的U盘，可以识别得到
不过FAT32传文件时有4G大小限制，有空试试exFAT或别的格式，exfat也可以，只是我想玩的proot容器的Debian系统无法运行，暂时不清楚根本原因

Alist

需要先安装termux
termux-app_v0.118.0+github-debug_armeabi-v7a.apk

pkg install alist
alist server记得看一下日志输出的admin用户默认随机密码，等会进去改成自己的密码
测试了一下是可以读写的USB存储设备的。

可以试试USB外接一些大存储的硬盘，
USB移动硬盘
USB的nv硬盘盒子+硬盘目前我用的这个方案当缓存盘用，省电轻便，准备有空再来两个归档盘

termux-media-play 床头音频播放器

这玩意不是有3.5mm的音频输出孔嘛，别浪费了，插个耳机或者喇叭音响变身床头音乐盒
直接播放文件夹内所有音乐termux-media-play play xxx.mp3，命令只能接一个文件名，可以配合红外手机遥控调节音量

sox方案也可以播音频，但是在我使用一段时间后sox报错了，需要重启才能正常
pkg install sox
命令可以接多个文件名
play ./*/*.mp3
sox: Sorry, there is no default audio device configured
或许这个依赖有用，但是我已经局域网了没机会尝试，apt install libsox-fmt-all

语音引擎

termux-tts-engines返回空，那就是没有内置语音引擎？感觉安装几个
安装后要去安卓系统设置无障碍那里启用，小米和科大讯飞居然是一样的声音

termux-tts-speak msg……一般用这个就可以朗读msg了
如果想切换现有引擎的话可以
termux-tts-speak -e "com.xiaomi.mibrain.speech" '大家转载文章注意标明文章出处啊'

自启动配置

termux自启动和启动项配置
vim ~/.bashrc

chmod +x ~/.bashrc
下面的命令需要配合termux-boot
cp ~/.bashrc ~/.termux/boot/autoTask
chmod +x ~/.termux/boot/autoTask
