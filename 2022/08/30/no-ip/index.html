免费的DDNS

设置域名，绑定ip，通过官方接口更新动态ip，实现动态域名解析DDNS
我选了个比较短的ddns.net，往后不出意外的话主机名就一直用ayao.ddns.net了。

ipv6更新接口

官方的文档有小错误，还是按这个模板来吧
主要是Authorization，需要填的是Basic (username:password)toBase64，即填入Basic 空格 (用户名:密码)的base64编码

接口URL

https://dynupdate.no-ip.com/nic/update?hostname=ayao.ddns.net&myipv6=2100:8a5c:1425:eff0:d2a4:6fff:fe68:de0f

请求方式

GET

Content-Type

multipart/form-data

请求Header参数

参数名 示例值 参数类型 是否必填 参数描述
Host dynupdate.no-ip.com Text 是 -
Authorization Basic base64 Text 是 Basic (username:password)toBase64
User-Agent Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/104.0.5112.102 Safari/537.36 Edg/104.0.1293.70 Text 是 -

请求Query参数

参数名 示例值 参数类型 是否必填 参数描述
hostname ayao.ddns.net Text 是 -
myipv6 2100:8a5c:1425:eff0:d2a4:6fff:fe68:de0f Text 是 -
