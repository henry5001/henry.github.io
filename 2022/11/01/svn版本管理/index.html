svn版本管理

SVN是subversion的缩写，是一个开放源代码的版本控制系统，通过采用分支管理系统的高效管理，简而言之就是用于多个人共同开发同一个项目，实现共享资源，实现最终集中式的管理。

svn的安装和配置

以linux系统为例
svn安装apt-get install subversion

创建仓库文件夹，首先建立svn文档的根目录 svnrepo, 然后创建一个项目仓库的文件夹 uapp
mkdir /svnrepo
mkdir /svnrepo/uapp
将uapp作为仓库
svnadmin create /svnrepo/uapp
启动 svn 服务，并指定 svn 数据存储的根目录
svnserve -d -r /svnrepo

svnWebUi可视化后台管理

svnWebUi运行起来后可以通过浏览器很方便地配置用户名密码，仓库，用户，组等权限。
需要提前安装Java
把jar包运行起来即可，注意启动项设置
nohup java -jar -Dfile.encoding=UTF-8 ~/svnWebUI/svnWebUI.jar --server.port=6060 --project.home=~/svnWebUI/ > ~/logs/svnUI.log &
